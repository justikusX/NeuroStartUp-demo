## Демонстрация преимуществ Git

Этот проект создан, чтобы показать ключевые преимущества использования системы контроля версий:

1.  **Хранение истории изменений**
    *   Каждое изменение в проекте фиксируется. Вы всегда можете посмотреть, кто, когда и зачем внес правки.
2.  **Видимость и авторство каждого изменения**
    *   Git записывает автора каждого коммита, обеспечивая полную прозрачность разработки.
3.  **«Резервная копия» проекта**
    *   Хранение кода на GitHub — это не только удобный доступ, но и надежная резервная копия, защищенная от потерь на локальном компьютере.
